# 혼클로드 (honclwd)

비개발자가 **혼자서도 안전하게** 만들 수 있도록 돕는 Claude Code 워크플로우 플러그인입니다.
코드를 직접 읽지 못해도, 안전장치와 쉬운 한국어 설명이 옆에서 받쳐줍니다.

## 무엇을 해주나요

- **작업 시작 카드** — 만들기 전에 "무엇을·어디까지" 합의하고 시작합니다.
- **검증 게이트** — 계획과 코드를 *적대적으로* 검수해(plan-validator / pr-reviewer) 위험을 미리 잡습니다.
- **실제 구동 검증** — 화면·앱을 코드로만 끝내지 않고, 격리된 테스트 환경에서 실제로 띄워보고 눌러봅니다.
- **쉬운 한국어 요약** — 기술 결과를 "지금 무엇을 했고, 잘못되면 어떤 일이 생기는지"로 풀어 설명합니다.
- **멈춤 규칙** — 되돌리기 어려운 일(삭제·배포 등) 앞에서는 먼저 확인을 받습니다.
- **레퍼런싱** — 기획 중 비슷한 사례·경쟁사를 찾아 정리합니다.
- **제품 지도** — 기능 목록과 화면 구조(IA)를 자동으로 정리·갱신합니다.
- **디자인 시스템 정합성** — 새 화면이 기존 디자인 규칙과 맞는지 보고, 새 패턴은 규칙으로 등록 제안합니다.

## 설치

```
/plugin marketplace add <이 저장소>
/plugin install honclwd
```

설치하면 **새 세션이 시작될 때 워크플로우가 자동으로 켜집니다.** 따로 설정 파일을 편집할 필요가 없습니다.

## 함께 설치되는 것 (중요)

이 플러그인은 [Superpowers](https://github.com/obra/superpowers)의 방법론 스킬(아이디어 정리·계획·디버깅 등)을 사용합니다.
- Superpowers는 **의존성으로 자동 설치**됩니다(직접 따로 설치하지 않아도 됩니다).
- 안전을 위해, 설치 전 **`claude-plugins-official` 마켓플레이스가 등록**되어 있는 것을 권장합니다.
- **Claude Code 최신 버전(권장 v2.1.143 이상)** 에서 자동 설치가 안정적으로 동작합니다. 구버전이면 Superpowers가 자동 설치되지 않을 수 있으며, 그 경우 워크플로우가 "Superpowers가 필요하다"고 안내합니다.

## 구성

- `rules/operating-rules.md` — 워크플로우 본체(세션 시작 시 자동 적용)
- `skills/referencing/SKILL.md` — 기획 중 레퍼런스 조사
- `skills/product-map/SKILL.md` — 기능 명세·IA 구조 정리/갱신
- `skills/design-system/SKILL.md` — 디자인 규칙 검증·갱신
- `agents/plan-validator.md` — 계획 검수 게이트
- `agents/pr-reviewer.md` — 코드 검수 게이트
- `agents/code-implementer.md` — 단순·기계적 코드 구현 담당
